# Sicherheit

Valiora verarbeitet personenbezogene Daten (Google-Login, persönliche Einkaufslisten).
Verantwortungsvolle Meldungen von Sicherheitsproblemen sind uns deshalb wichtig.

## Eine Sicherheitslücke melden

**Bitte erstelle für Sicherheitsprobleme kein öffentliches Issue.**

Melde sie stattdessen vertraulich:

- **GitHub Security Advisories** — über den Button *„Report a vulnerability"* im Tab
  **Security** des betroffenen Repositories (bevorzugt), oder
- **E-Mail** an Paul Gradischnig: `pgradischnig@yahoo.com`

Hilfreich sind: eine kurze Beschreibung samt Auswirkung, Schritte zum Nachstellen und
die betroffene Komponente.

Wir melden uns so schnell wie möglich zurück, halten dich zum Fix auf dem Laufenden
und nennen dich auf Wunsch nach der Behebung als Finder:in.

## Hinweis

Valiora ist ein Schülerprojekt in aktiver Entwicklung und (noch) kein Produktivdienst
mit Bug-Bounty-Programm. Meldungen werden nach bestem Bemühen geprüft und behoben.
